Guide de mise en forme

Vous n’avez besoin d’indiquer que les valeurs que vous souhaitez modifier. Toutes les valeurs manquantes utiliseront les paramètres par défaut ci-dessous.

  • font_size : Ajuste la taille du texte (Défaut : 13).
  • font_color : Change la couleur du texte (Défaut : "#000000").
  • font_family : Change la police du texte (Défaut : autre choix sont "Times-Roman" ou "Courier").
  • alignment : Peut être "left", "center" ou "right" (Défaut : "left").
  • line_spacing : Contrôle l’espacement entre les lignes (Défaut : 1.2).
  • space_Before : Espacement avant une section (Défaut : 12).
  • space_After : Espacement après une section (Défaut : 12).

✅ **Exemple** : Si vous souhaitez uniquement modifier la taille du texte, entrez :

{
    "font_size": 16
}

⚠️ Toutes les autres valeurs resteront inchangées.

⚠️ Important : Ne supprimez pas complètement le {} !

Si vous ne souhaitez apporter aucune modification, laissez le champ vide ou utilisez {}, cela sera géré automatiquement.
