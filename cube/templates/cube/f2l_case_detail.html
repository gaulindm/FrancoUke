{% if method == 'OLL' %}
Conseils OLL
  • Reconnaître le pattern avant de chercher l'algorithme
  • Pratiquer la reconnaissance en regardant seulement 2 faces
  • Certains OLL ont plusieurs algorithmes — choisissez le vôtre
  • Les cas symétriques peuvent être résolus avec le miroir
{% elif method == 'PLL' %}
Conseils PLL
  • Cherchez les "headlights" (deux stickers identiques côte à côte)
  • Identifiez les blocs 1x2 formés par les coins
  • Commencez par les PLL les plus fréquents
  • Pratiquez l'AUF pour aligner correctement
{% else %}
Conseils F2L
  • Repérez la paire coin + arête avant de commencer
  • Essayez de comprendre le pourquoi de chaque mouvement
  • Apprenez d'abord les 4 cas de base, puis les variantes
  • La vitesse vient avec la mémoire musculaire — patience!
{% endif %}
Comment pratiquer
  1. Regardez l'animation plusieurs fois
  2. Utilisez ▶| pour avancer mouvement par mouvement
  3. Décomposez en groupes de 3–4 mouvements
  4. Pratiquez lentement, augmentez la vitesse progressivement
  5. Faites 20–30 répétitions pour la mémoire musculaire
